# プロジェクト憲法：店舗営業時間案内アプリ

このドキュメントは、本プロジェクトの仕様、技術スタック、および開発フェーズを定義するものです。

## 1. アプリの目的

HUB・82 等の店舗（約 100 店舗）の営業時間を、ユーザーが日付や場所から素早く確認できるようにする。

## 2. ターゲット・利用シーン

- **ターゲット**: 外出先や利用直前に店舗の営業時間を知りたいユーザー。
- **利用シーン**: 「今から行く店は開いているか？」「明日、富山の店は何時に閉まるか？」といった疑問を数タップで解決する。

## 3. MVP（最小構成）の範囲

- スプレッドシート由来の JSON データを取得・表示する。
- 地域・都道府県・市区町村・キーワードによる絞り込み。
- 向こう 7 日間の営業時間予報を表示。
- 公式サイト各店舗ページへのリンク。

## 4. 技術スタック

- **フロントエンド**: React 19 / TypeScript / Vite
- **スタイリング**: Vanilla CSS (Tailwind CSS 併用)
- **データソース**: GitHub Pages 上の JSON ファイル
- **更新周期**: 毎日午前 7:00 更新

## 6. 開発環境

### ローカルサーバー

- **開発サーバー起動**: `npm run dev`
- **ローカルURL**: http://localhost:3000/
- **ネットワークURL**: http://192.156.1.106:3000/ (同一ネットワーク内の他デバイスからアクセス可能)
- **ポート**: 通常は3000番（使用中の場合は自動的に3001、3002等に変更）

## 7. 特記事項

- **検索の挙動**: キーワードが入力されている場合は、全店舗を検索対象とする。キーワードが空の場合は、選択されている地域（初期値：関東）の店舗を表示する。
- **並び順**: 都道府県順（都道府県内は市区町村・五十音順）で表示する。
